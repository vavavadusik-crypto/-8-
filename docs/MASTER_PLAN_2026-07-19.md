## 1. EXECUTIVE SUMMARY

**Продукт.** Hermest Board — AI-студия контента: пользователь даёт тему → система исследует источники → строит карточки с фактами → сценарий → раскадровку → генерирует настоящую русскую озвучку и визуалы → рендерит настоящий MP4 (16:9 + 9:16) → человек одобряет → publish pack. Wedge-позиционирование: «explainable content production board» для образовательного/технического контента. Не «Miro-убийца» и не «автономная студия без человека».

**Где мы (проверено фактически 2026-07-19).** Фундамент отличный: доска, безопасность (XSS/CSP, argv-санитизация, sealed candidates), детерминированный FFmpeg-рендер MP4, локальный worker, каталог 44 провайдеров, 122 unit-теста, живой деплой на Vercel. **Но ядро ценности отсутствует**: русского голоса нет (flite = English-only), генерации изображений нет (0%), автоматизации тема→карточки нет (10%), долговременного хранилища нет (localStorage/json-file), биллинга нет. Текущие рекламные ролики («Дмитрий», «Светлана») — запись экрана, что прямо противоречит North Star.

**Стратегия.** Money-first: сначала достроить ядро ценности (голос → визуалы → автоматизация), сразу же продать его руками (done-for-you и concierge private beta), и только потом достраивать полный self-serve SaaS с биллингом. Голос и субтитры — сразу мультиязычные (старт: RU/EN/ES/DE/FR; режим «любой язык» через премиум-TTS): рынок голосового видео глобальный, и охват языков — главное оружие конкурентов (Fliki: 75+ языков). Автопубликация (OAuth) и аналитика — ПОСЛЕ первых денег.

**Конкуренция (проверено 2026-07-19).** Прямые аналоги (Fliki, Pictory, InVideo, Lumen5, AutoShorts, StoryShort, Revid) продают «минуты видео» из стоковых картинок без источников. Никто из них не делает research-grounded контент с цитатами, BYOK-экономику и прозрачный конвейер с human approval — это наш клин. Детали и позиционирование — раздел 2.5.

**Путь к деньгам.**
- Первые деньги (~2–3 недели, после P4): done-for-you генерация видео клиентам + concierge private beta (ручной инвойс, 5–10 клиентов).
- Полноценный платный запуск (~6–8 недель, после P6): тарифы $0 / $19 / $39 / $99 + usage packs.

**Общая готовность:** ~35–40% как SaaS; ~70% как локальная студия. Критический путь: **P1 (русский голос) → P2 (визуалы) → P3 (автоматизация)** — это и есть продукт.

| StoryShort / Revid.ai | 39+ | faceless-серии, персонажи, быстрый рендер | нет источников; дороже нашего Creator $19 |

**Чем обходим (клин, всё уже в архитектуре):**

1. **Research-grounded**: карточки-источники с цитатами, lineage и подсветкой противоречий — ни у одного конкурента; ниша образовательного/технического/новостного контента, где важна достоверность.
2. **BYOK-экономика**: пользователь приносит свои ключи → мы не перепродаём «минуты» с наценкой. Оффер, которого нет у конкурентов: «свои ключи — генерируй без лимита минут».
3. **Прозрачный конвейер + human approval**: каждый шаг виден и редактируем (у конкурентов — чёрный ящик «промпт→видео»); immutable candidates и provenance/rights manifest — аргумент для брендов и агентств.
4. **Локальный рендер**: privacy + реально бесплатный Free-тариф (у конкурентов free = watermark и огрызок минут).
5. **Мультиязычные editions**: один проект → те же сцены на N языках. У Fliki много голосов, но «проект→языковые версии» как first-class сущность — наш ход.
6. **Цена**: Creator $19 ниже Fliki (~28), Pictory (25), StoryShort (39) при большем контроле.

**Честно о слабостях:** по числу языков Fliki впереди (75+ vs наши 5 native + 29+ через ElevenLabs); по аватарам не конкурируем сознательно (красный океан Synthesia/HeyGen); авто-постинга у нас нет до R6 — продаём publish pack и контроль, а не «постинг за тебя».

## 3. ЦЕЛЕВОЙ КОНВЕЙЕР (North Star, неизменен)

```
тема (естественный язык)
  → AI Director: уточняющие вопросы (с дефолтами)
  → research: публичные источники + BYOK-модели
  → source/fact cards с цитатами и lineage (противоречия подсвечены)
  → outline → сценарий → раскадровка (structured contracts, ручной override везде)
  → озвучка на языке проекта: реальный аудиофайл (Piper локально — RU/EN/ES/DE/FR и др.; ElevenLabs BYOK — 29+ языков, режим «любой язык»)
  → визуалы: сгенерированные изображения (FAL BYOK) + Commons
  → FFmpeg: MP4 16:9 (1920x1080) + 9:16 (1080x1920), SRT, обложки, loudnorm
  → семантические shorts
  → мультиязычные editions: один проект → те же сцены на N языках (перевод сценария structured-контрактом + локализованные озвучка и SRT)
  → manifest (хеши, lineage, QC, rights/provenance)
  → human approval (immutable candidate)
  → publish pack (экспорт сразу; автопубликация — после OAuth review)
```

Главный продуктовый тест каждой задачи: **приближает ли она этот путь**. Если нет — задача не в приоритете.

## 6. АРХИТЕКТУРНЫЕ ИНВАРИАНТЫ (для премиум-качества, проверяются на каждом ревью)

1. **Слои**: Domain (storyboard/pipeline — чистые функции) → Application (jobs/candidates) → Infrastructure (FFmpeg/TTS/провайдеры/storage) → Presentation (UI/API). Зависимости только внутрь. Новые адаптеры — только через существующие порты (`tts.js`-контракт, capability router).
2. **Детерминизм**: одинаковый вход → одинаковый manifest/хеши. Всё недетерминированное (генерация) фиксируется в manifest (model, prompt-hash, seed где есть).
3. **Fail-closed**: отсутствие QC/rights/provenance = блок, никогда не «пропустим». Прецеденты уже в коде (`requirePassedRenderQc === true`).
4. **Секреты**: только BYOK-vault/`${ENV}`; ни в manifest, ни в логах, ни в argv (санитизация carrier-форм уже есть — не ослаблять).
5. **Деньги под контролем**: любой платный вызов — предварительная оценка + жёсткий бюджет + UsageRecord. Retry после платного side effect — только через reconciliation.
6. **Честность статусов**: словарь VERIFIED/PARTIAL/MOCKED/MISSING/BLOCKED; заявление без доказательства не повышается. Документы обновляются в том же PR, что и код.
7. **TDD**: сначала падающий узкий тест, потом код, потом полный гейт. Без тестов задача не существует.

1. **Канон для исполнителей — в репозитории**: `docs/MASTER_PLAN_2026-07-19.md` (репо-копия этого плана, коммитится и пушится на GitHub). Desktop-копия — витрина для Вадима; при расхождении права репо-копия.
2. **`docs/EXECUTION_STATE.md` — единственный источник «где мы сейчас»**. Обязательные поля: ACTIVE PHASE/TASK, STATUS (IN_PROGRESS/BLOCKED/DONE), LAST COMMIT (SHA + суть), NEXT ACTION (буквально следующая команда или файл), UNCOMMITTED (что могло остаться недописанным), BLOCKERS. Обновляется в НАЧАЛЕ каждой задачи и ПОСЛЕ каждого коммита — это часть Definition of Done, не «потом».
3. **Коммиты — маленькие и частые**: после каждого зелёного узкого теста. Рабочее дерево не остаётся грязным дольше одной подзадачи. Полный `npm run check` — перед закрытием задачи (не перед каждым коммитом).
4. **Push после каждой завершённой задачи**: GitHub — резервная копия; смерть машины/сессии не теряет работу.
5. **Возобновление одной фразой.** На «продолжи Hermest Board» любой агент выполняет: `cat docs/EXECUTION_STATE.md` → `git log -5 --oneline` → `git status --short`. Если дерево чистое — продолжать с NEXT ACTION. Если грязное — обрыв случился посреди записи: сверить каждый изменённый файл с NEXT ACTION/UNCOMMITTED, дописать или откатить файл (не весь worktree!), закоммитить, обновить EXECUTION_STATE, продолжить.
6. **Никогда**: `git reset --hard` / `git clean` в каноничном worktree; force-push; правки без записи в EXECUTION_STATE при завершении.
7. **Эстафетное сообщение (обязательно для КАЖДОГО агента).** Когда лимит сессии подходит к концу (<20% или первое предупреждение) — ПЕРЕД продолжением любой задачи: (а) обнови `docs/EXECUTION_STATE.md`; (б) перепиши `docs/NEXT_CLAUDE_MESSAGE.md` и его копию на Рабочем столе (`HERMEST_BOARD_СООБЩЕНИЕ_СЛЕДУЮЩЕМУ_КЛОДУ.md`): короткое сообщение следующему агенту — где остановился, что осталось незакоммиченным, какая первая команда, какие грабли этой сессии; (в) закоммить и запушь. ## 13. ИТОГ ОДНИМ АБЗАЦЕМ

Фундамент Hermest Board — инженерно сильный и отревьюенный, но продавать пока нечего: нет живого голоса, нет генерации визуалов, нет автоматизации «тема→видео». План: P0 слить и задеплоить → P1 Piper/ElevenLabs мультиязычный голос (русский первым; матрица RU/EN/ES/DE/FR, «любой язык» через ElevenLabs 29+) → P2 FAL-визуалы с моушеном → P3 AI Director «одна кнопка» + мультиязычные editions одного проекта → P4 продукт сам генерирует свою рекламу на 4 языках (и открываются done-for-you продажи) → P5 Supabase-ядро → P6 Paddle-биллинг с тарифами $0/$19/$39/$99 и concierge-бетой. Против Fliki/Pictory/AutoShorts бьём не «минутами», а research-grounded контентом с цитатами, BYOK-экономикой («свои ключи — без лимита минут») и прозрачным конвейером с human approval. Первые деньги реалистичны через 2–3 недели (услуги), платный self-serve — через 6–8 недель. Каждый шаг — TDD, зелёный `npm run check`, независимое ревью, честные статусы.
